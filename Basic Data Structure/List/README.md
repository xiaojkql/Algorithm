## 列表

因为列表的底层不再是原始的C++所有的数据类型。

因为列表是通过地址互相指向，所以常常操作的是节点的地址。

### 1 数据成员

包含三项

头结点：header，用来记录列表的头。列表所必须的。

尾节点：trailer，用来记录列表的头。列表所必须的。

节点的数量：_size，用来记录除头尾节点的数量的项。
