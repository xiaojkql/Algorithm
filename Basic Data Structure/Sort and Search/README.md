### 5 基本排序算法

效率低但容易编写，效率高但很难编写，两者中进行取舍。

#### 5.1 选择排序

“选择”的意思就是选择第排序后第 i 项的的值，然后与当前的值进行交换，从而第 i 项得到排序后应该得到的正确的值。

复杂度：比较次数：(n-1+(n-2)+(n-3)+...+1 = O(n^2)，交换次数:O(n),总复杂度：O(n2)。
